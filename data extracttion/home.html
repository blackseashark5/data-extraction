PDF file (.PDF)

PDF ( Portable Document Format ) is a versatile document format that allows the exchange of electronic documents regardless of operating systems, application/software or hardware. Each PDF file describes its content by placing characters on a 2-D coordinate plane. Besides plain text & images, PDF’s may also contain structured key-value pairs, annotations, fields, layers, videos & more.

CSV file (.CSV )

CSV ( comma-separated values ) is a delimited text file that represents data in a tabular format. In a CSV, each column value is separated by a comma and each row is represented in a new line. Also referred to as a flat file format, CSV’s can be opened on all spreadsheet or table-oriented applications. CSV os a popular format for exchanging data between applications.

Frequently Asked Questions

How does Nanonets work?
Nanonets uses OCR technology that leverages AI & ML capabilities to extract tables from PDFs, scanned files & images. To convert a PDF document to the editable CSV format, it is important to capture relevant data stored in tabular structures on the document, making it suitable for further
editing or data processing.
How does Nanonets rename PDF files based on content?
Nanonets leverages advanced Machine Learning (ML) techniques to intelligently rename PDF files based on their content. By using Optical Character Recognition (OCR), it can read and comprehend the content within your PDF files. After the OCR process, it applies Natural Language Processing (NLP) techniques to understand the context and determine the most relevant parts of the document. Based on this information, Nanonets generates a suitable name that represents the content of the file. It's a convenient feature for organizing large volumes of PDFs, ensuring you can locate and identify the necessary files with ease.
